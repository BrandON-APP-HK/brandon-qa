# 21. Account Profile Edit

## Scope

My Details: display name, phone/country code, gender, birthdate, email, and password entry visibility.

## Preconditions

- User is logged in.

## Scenario 21.1: Edit display name

- Open **Me** / Account.
- Expand **My Details**.
- Tap edit for name.
- [ ] Name field should become editable.
- Enter fewer than 3 characters.
- [ ] Validation should reject short name.
- Enter valid name and blur.
- [ ] Name should save and remain displayed.

## Scenario 21.2: Edit phone and country code

- Tap edit for contact.
- [ ] Country code selector and phone input should become editable.
- Select a country code.
- [ ] Country code should update.
- Enter numeric phone value and blur.
- [ ] Phone should save.
- Try non-numeric input.
- [ ] Input should be constrained by field behavior.

## Scenario 21.3: Edit gender and birthdate

- Tap edit for gender.
- Select a gender value.
- [ ] Gender should save and display selected value.
- Tap edit for birthdate.
- [ ] Date picker should open.
- Select future date.
- [ ] Future date should not be accepted.
- Select valid past/current date.
- [ ] Birthdate should save and picker should close.

## Scenario 21.4: Email and change-password visibility

- Open My Details.
- [ ] Email should display read-only.
- For password-login user, check change password.
- [ ] Change Password row should be visible.
- For Google/Apple user, check change password.
- [ ] Change Password should be hidden or unavailable.
