# 28. Static Legal Help Pages

## Scope

Terms, privacy policy, help center, and learn-about pages.

## Preconditions

- App can reach page content APIs.

## Scenario 28.1: Terms from login modal

- Open login bottom sheet.
- Tap **Terms** link.
- App should route to **Terms**.
- Terms document should be visible.
- Page should be scrollable.
- Tap close/back.
- App should return to the previous screen or **Home**.

## Scenario 28.2: Privacy policy from login modal

- Open login bottom sheet.
- Tap **Policy** link.
- App should route to **Privacy Policy**.
- Privacy policy document should be visible.
- Page should be scrollable.
- Tap close/back.
- App should return to the previous screen or **Home**.

## Scenario 28.3: Help Center

- Open drawer.
- Tap **Help Center**.
- Help Center page should open.
- Header title should show Help Center.
- Page sections should load and be ordered as configured.
- Expand a section.
- Section title, subtitle, and body should be visible.
- Collapse the section.
- Section body should hide.
- Tap close/back.
- App should dismiss to **Home**.

## Scenario 28.4: Learn About

- Open drawer.
- Tap **About**.
- Learn About page should open.
- Header title should show Learn About/About content.
- Page sections should load and be ordered as configured.
- Expand and collapse a section.
- Expanded content should show and hide correctly.
- Tap close/back.
- App should dismiss to **Home**.
