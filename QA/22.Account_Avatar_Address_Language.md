# 22. Account Avatar Address Language

## Scope

Avatar upload, address search/save, and language preference.

## Preconditions

- User is logged in.
- Camera/photos permissions can be reset.

## Scenario 22.1: Change avatar

- Open Account.
- Tap avatar/edit overlay.
- [ ] Source options should include camera, gallery, and cancel.
- Choose camera or gallery and select image.
- [ ] Avatar area should show loading while uploading.
- [ ] Success toast should appear on upload success.
- [ ] New avatar should display after refresh.

## Scenario 22.2: Avatar permission recovery

- Deny camera permission and try camera avatar.
- [ ] Alert/recovery should show Retry.
- Block camera permission and try again.
- [ ] Open Settings and Choose from Gallery should be available.
- Deny/block gallery permission and try gallery avatar.
- [ ] Retry or Open Settings should be shown depending on permission state.
- Test limited gallery access if available.
- [ ] Limited access information should be shown.

## Scenario 22.3: Address search and save

- Expand Address section.
- Type one character in building/address search.
- [ ] No search should run.
- Type at least two characters.
- [ ] Address search should run.
- Select a suggested address.
- [ ] Building, street, town, and state fields should populate where returned.
- Fill required fields: addressee, building, street, town, state.
- Tap confirm/save.
- [ ] Loading should show while saving.
- [ ] Success indicator should appear after save.

## Scenario 22.4: Language preference

- Expand Preferences.
- Open language selector.
- [ ] English and Chinese options should be visible.
- Select Chinese.
- [ ] App language should change immediately.
- [ ] Selected language should save to profile.
- Select English.
- [ ] App language should switch back and save.
