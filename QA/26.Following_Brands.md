# 26. Following Brands

## Scope

Following Brands tabs, filtering, brand navigation, follow/unfollow, and refresh.

## Preconditions

- Brand data includes brands with and without active promotions.
- Test as logged-in user; direct logged-out route can also be checked.

## Scenario 26.1: Tabs and filtering

- Open drawer > **Following Brands**.
- With Promotions tab should be selected by default.
- Only brands with active promotions should show.
- Switch to Without Promotions.
- Only brands without active promotions should show.
- Leave and return to page.
- Selected tab should reset to With Promotions.

## Scenario 26.2: Brand navigation

- On With Promotions, tap a brand row/logo with active promotion.
- Brand campaign page should open.
- If the brand has multiple active campaigns, the newest active campaign should open.
- Return and switch to Without Promotions.
- Tap a brand row/logo.
- It should not navigate when no active campaign is available.

## Scenario 26.3: Follow and unfollow

- Tap an unfilled star.
- Star should update locally.
- Push notification registration may be requested.
- Follow request should complete.
- Tap a filled star.
- Star should update locally.
- Unfollow request should complete.
- Pull to refresh.
- Brands should refetch and temporary pressed state should clear.
- Followed brands should sort above non-followed brands.

## Scenario 26.4: Logged-out follow prompt

- Log out and open Following Brands by deep link/direct route if possible.
- Tap a star.
- Login bottom sheet should open instead of following/unfollowing.
