# 11. Saved Liked Deals

## Scope

Liked-only bottom tab behavior and liked feed filtering.

## Preconditions

- Logged-in test user has at least one liked deal and one unliked deal available.

## Scenario 11.1: Guest likes tab is guarded

- Start logged out.
- Tap **Likes** tab.
- [ ] Login bottom sheet should open.
- [ ] Liked-only mode should not toggle.
- [ ] Likes icon should remain inactive.

## Scenario 11.2: Toggle liked-only mode

- Log in.
- Open Home.
- Tap **Likes** tab.
- [ ] Liked-only mode should turn on.
- [ ] Likes icon should change to active state.
- [ ] Feed should show liked deals only.
- Tap **Likes** tab again.
- [ ] Liked-only mode should turn off.
- [ ] Feed should return to normal results.

## Scenario 11.3: Unlike from liked-only feed

- Log in and turn liked-only mode on.
- Tap heart on a liked deal.
- [ ] Deal should become unliked.
- [ ] Liked-only feed should refresh.
- [ ] Unliked item should disappear or no longer appear as liked after refresh.

## Scenario 11.4: Liked-only with search and sort

- Turn liked-only mode on.
- Change Home sort to Rating.
- [ ] Feed should remain liked-only while using Rating sort.
- Open Search and submit a query.
- [ ] Search results should respect liked-only mode.
- Turn liked-only mode off.
- [ ] Search/Home results should no longer be liked-only filtered.
