# 2. User Login Apple iOS

## Scope

Apple authentication from the shared login bottom sheet on iOS.

## Preconditions

- Test on iOS.
- Tester is logged out.
- Apple ID is available.

## Scenario 2.1: Apple button visibility on iOS

- Start the iOS app.
- Tap **Me** while logged out.
- [ ] Login bottom sheet should open.
- [ ] **Continue with Apple** should be visible.
- [ ] **Continue with Google** should be visible.
- [ ] **Continue with Email** should be visible.

## Scenario 2.2: Successful Apple login

- Open the login bottom sheet on iOS.
- Tap **Continue with Apple**.
- [ ] Native Apple authentication should open.
- Complete Apple authentication.
- [ ] Login bottom sheet should close.
- [ ] App should return to `/home`.
- [ ] Protected tabs and auth-only drawer items should become available.

## Scenario 2.3: Apple button hidden on Android

- Start the Android app.
- Open the login bottom sheet.
- [ ] Apple login button should not be displayed.
- [ ] Google and Email options should remain available.
