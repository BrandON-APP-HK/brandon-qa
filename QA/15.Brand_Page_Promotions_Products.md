# 15. Brand Page Promotions Products

## Scope

Brand campaign page, promotion info/rules, contact/reserve links, products, tags, and pagination.

## Preconditions

- Brand campaign exists with logo, images, description, promotion fields, and products.

## Scenario 15.1: Brand page core content

- Open a brand campaign from a promo block or map/list.
- Brand page should open.
- Header/top bar should be visible.
- Brand logo should display.
- Follow button should display.
- Campaign images should display and swipe when multiple images exist.
- Brand name and description should display.
- Expand long description if available.
- Description should expand/collapse correctly.

## Scenario 15.2: Promotion information and rules

- On brand page, find promotion section.
- Promotion title/details should be visible when configured.
- Tap promotion info/rules control.
- Promotion rules modal should open.
- Rule text should match campaign fields.
- End date should display only when campaign has an end date.
- Close modal.
- Brand page should remain visible.

## Scenario 15.3: Contact and reserve links

- Open a campaign with contact/reserve enabled and URLs configured.
- Tap contact button.
- External URL or supported app should open.
- Return to app and tap reserve button.
- Reserve URL should open.
- Open a campaign without those URLs.
- Missing/disabled contact and reserve buttons should not appear as empty controls.

## Scenario 15.4: Product tags and list

- Open product section.
- **All** tag should be selected by default.
- Product tags should include tags derived from products.
- Select a non-All tag.
- Products should filter/reload for that tag.
- Select **All**.
- Other tags should clear and all products should return.
- Product cards should show image, name, description, price/old price/discount where present.
- Scroll near bottom.
- Next product page should load when more products exist.

## Scenario 15.5: Brand short page

- Open a brand short route/context if available.
- Short brand page should show compact campaign information.
- Tap navigation/CTA to full brand page.
- Full brand page should open for the same campaign.
