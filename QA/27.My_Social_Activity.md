# 27. My Social Activity

## Scope

My Social Activity feed, refresh, pagination, liked-only interaction, details, and report.

## Preconditions

- User is logged in.
- User has social activity feed data or an empty-state account.

## Scenario 27.1: Open feed

- Open drawer > **My Social Activity**.
- [ ] Page header should show My Social Activity.
- [ ] Loading indicator should show while initial list is empty and loading.
- [ ] Social deal blocks should render when returned.
- [ ] Promo blocks should render when returned.

## Scenario 27.2: Refresh and pagination

- Pull to refresh.
- [ ] Refresh control should appear.
- [ ] Feed should refetch.
- If Distance sort is active, continue.
- [ ] Location should reload before/while feed refetches.
- Scroll near the bottom.
- [ ] More feed items should load when more pages exist.

## Scenario 27.3: Deal detail and flag

- Tap a social deal image.
- [ ] Social deal details modal should open in my-social context.
- Close details.
- Tap flag/report.
- [ ] Flag modal should open in my-social context.
- Select reason and submit.
- [ ] Modal should close and item should be flagged.

## Scenario 27.4: Liked-only interaction

- Turn liked-only mode on from bottom tab.
- Open My Social Activity.
- [ ] Feed should apply liked-only filter.
- Turn liked-only off.
- [ ] Feed should return to normal social activity results.
